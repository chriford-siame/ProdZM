# ProdZM - An Ecommerce Website

ProdZM is a n ecommerce website which allows customers all over the world to purchase products on a large scale and be able to sale those particular products under some conditions/permissions.

## Tech Stacks Used
- React JS
- Typescript
- TailwindCSS

## Project Launch Steps
```javascript
1. clone the project
2. cd ProdZM
3. npm install
4. npm run start 
```
The project by default runs on  [http://localhost:3000](http://localhost:3000)

## Installation Requirements
- NodeJS 16+
